## How to Deploy:

### Option 1: Upload to Web Hosting (Recommended)

1. **Get a domain name:**
   - Recommended: techtechkievisuals.com or techtechkievisuals.co.uk
   - Register through Namecheap, GoDaddy, or any domain registrar

2. **Get web hosting:**
   - Cheap options: Hostinger (~£2/month), SiteGround, Bluehost
   - Or use free hosting: GitHub Pages, Netlify, Vercel

3. **Upload files:**
   - Upload all 5 files to your hosting's public_html or www folder
   - Make sure they're in the root directory (not in a subfolder)

4. **Test:**
   - Visit your domain - you should see the homepage
   - Check all navigation links work

### Option 2: GitHub Pages (Free)

1. Create a GitHub account
2. Create a new repository named: `techtechkievisuals-website`
3. Upload all 5 files to the repository
4. Go to Settings → Pages
5. Select "main" branch and save
6. Your site will be live at: `yourusername.github.io/techtechkievisuals-website`
7. You can connect a custom domain later

### Option 3: Netlify (Free - Easiest)

1. Go to netlify.com
2. Sign up for free account
3. Drag and drop all 5 files into Netlify
4. Your site goes live instantly
5. Connect custom domain in settings
